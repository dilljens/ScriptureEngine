# Scripture Knowledge Engine — Glossary

| Term | Definition | Context |
|------|------------|---------|
| Gematria | Hebrew numerical value system — each letter has a number, words/phrases with equal values may be meaningfully related | Numerical connection layer |
| Strong's Number | Numeric identifier for every Hebrew and Greek word in the Bible (e.g., H7225 = ראשית "beginning") | Linguistic layer, lemma mapping |
| Morphology | Grammatical form of a Hebrew/Greek word (root, stem, tense, person, number, gender) | Linguistic layer |
| Chiasm | A-B-C-C'-B'-A' mirror structure where elements are repeated in reverse order | Structural layer |
| Parallelism | Poetic structure where adjacent lines relate (synonymous, antithetic, synthetic) | Structural layer |
| Inclusio | Literary "bookending" — same phrase opens and closes a passage | Structural layer |
| Intertextuality | How later texts quote, allude to, or echo earlier texts | Intertextual layer |
| Type-Antitype | Prophetic pattern (type) and its fulfillment (antitype) | Intertextual layer |
| Hapax Legomenon | A word that appears only once in the entire canon | Frequency layer |
| Midrash | Jewish interpretive tradition that re-uses earlier scripture in new contexts | Interpretive layer |
| JST | Joseph Smith Translation — revision of KJV Bible | Textual layer |
| LXX | Septuagint — Greek translation of the Hebrew Bible | Textual layer |
| WLC | Westminster Leningrad Codex — base Hebrew text used for OT | Data source |
